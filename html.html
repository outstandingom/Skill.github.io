Lesson 1: HTML Fundamentals

Building blocks of the web

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create webpages.

Key Features:
  • Defines the structure of web content
  • Uses tags to mark up text, images, and other elements
  • Works together with CSS and JavaScript

Basic HTML Tags

Headings & Paragraphs

<h1>Main Heading</h1>  <!-- Largest heading -->
<h2>Subheading</h2>
<h3>Smaller heading</h3>
<!-- Goes down to h6 -->

<p>This is a paragraph of text.</p>

Images & Links

<!-- Image -->
<img src="image.jpg" alt="Description of image">

<!-- Link -->
<a href="https://example.com">Visit Website</a>

Lists & Tables

<!-- Ordered List -->
<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

<!-- Unordered List -->
<ul>
  <li>Item one</li>
  <li>Item two</li>
</ul>

<!-- Table -->
<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Forms and Inputs

Forms collect user input and submit data to servers.

<form>
  <input type="text" placeholder="Your Name">
  <input type="email" placeholder="Your Email">
  <input type="submit" value="Submit">
</form>

Writing HTML on Mobile (Using Acode)

Complete Webpage Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is my first webpage.</p>
    
    <img src="image.jpg" alt="Sample image">
    
    <ul>
      <li>First item</li>
      <li>Second item</li>
    </ul>
    
    <form>
      <input type="text" placeholder="Your name">
      <input type="submit" value="Send">
    </form>
  </body>
</html>
