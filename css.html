Lesson 2: CSS Styling & Responsive Design

Make your website beautiful and work on all devices!

What is CSS?

CSS (Cascading Style Sheets) is the language used to style web pages.

Key Features:
  • Styles your HTML (colors, fonts, spacing, layout)
  • Keeps design separate from content
  • Works together with HTML and JavaScript

Basic CSS Properties

Selectors and Common Properties

/* Selectors */
p { }                  /* Element selector */
.class { }             /* Class selector */
#id { }                /* ID selector */

/* Colors & Fonts */
color: #333;           /* Text color */
background-color: #fff;/* Background */
font-family: Arial;    /* Font family */

/* Spacing */
margin: 10px;          /* Outer space */
padding: 15px;         /* Inner space */

/* Box Model */
border: 1px solid #ddd;

Layout with Flexbox & Grid

Modern CSS layout techniques for creating flexible designs.

Flexbox

Easy one-direction layout (row/column)

.container {
  display: flex;
  justify-content: center; /* Horizontal alignment */
  align-items: center;     /* Vertical alignment */
  gap: 20px;               /* Space between items */
}

CSS Grid

Create rows & columns for complex layouts

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr; /* Two equal columns */
  grid-gap: 15px;                /* Space between items */
}

Making Sites Responsive

Responsive design ensures your website works well on all devices.

Key Techniques:
  • Use relative units (%, em, rem)
  • Media queries adjust design for different screens
  • Flexbox and Grid are inherently responsive
/* Media Query Example */
@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
  
  .grid {
    grid-template-columns: 1fr; /* Stack on mobile */
  }
}

Bootstrap Framework

Bootstrap is a popular CSS framework that provides ready-made components and a responsive grid system.

<!-- Bootstrap Grid Example -->
<div class="container">
  <div class="row">
    <div class="col-md-6">Left Column</div>
    <div class="col-md-6">Right Column</div>
  </div>
</div>
Bootstrap Benefits:
  • Pre-designed components (buttons, cards, navbars)
  • Responsive grid system built-in
  • Consistent design across browsers
  • Large community and documentation

Summary

1 CSS styles and layouts your website

2 Flexbox & Grid help create modern, flexible designs

3 Responsive design ensures mobile friendliness

4 Bootstrap makes development faster and easier
