HTML Fundamentals

Lesson Notes

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

  • HTML describes the structure of a web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements are represented by tags
Tip: Always include the DOCTYPE declaration at the beginning of your HTML documents.

CSS Styling

Lesson Notes

CSS (Cascading Style Sheets) is used to style and layout web pages.

  • CSS describes how HTML elements should be displayed
  • CSS saves a lot of work by controlling multiple web pages at once
  • External stylesheets are stored in CSS files
Tip: Use external stylesheets for better maintainability.

JavaScript Basics & DOM Manipulation

Lesson Notes

JavaScript is a programming language that adds interactivity to your website.

  • JavaScript can validate data

Creating Website Using DeepSeek

GitHub Account Creation

Always give a detailed prompt to DeepSeek about what you want to use in your website. As much as possible, try to give reference code or design.

  • Make separate files for HTML, CSS and JavaScript
  • Divide your code into small parts to get more precise code
  • First give information about what you have created or what you want to create

GitHub Account & Hosting

GitHub Account & Hosting

GitHub is a platform where you can get free hosting for your website. You can use it for managing your code and see others' code easily. It's best for working in teams or even running on your mobile.

  • Make sure that your repository name ends with .github.io
  • index.html file is the home page of your site - it should be present in your repo
  • For creating public links, make sure you have YML in your repo

Firebase Integration

Firebase Integration

  • Choose test mode while creating project
  • For authentication, your GitHub name should be added in your auth settings like username.github.io
  • Make sure Firebase SDK has been added in your HTML code

Search Engine Optimization

Search Engine Optimization

  • Your sitemap should be added in your GitHub repo
  • Try to use keywords in your website in h1, h2 and h3 tags
  • Your most searchable keyword should be added in your title tag
  • Never delete XML file from GitHub repo

  • Learn about common threats and vulnerabilities
  • Explore security principles and best practices

Installing Termux

Lesson Notes

Termux is an Android terminal emulator and Linux environment app that allows you to run security tools on mobile devices.

  • Download and install Termux from F-Droid
  • Basic Termux commands and setup
  • Updating packages and installing dependencies
  • Setting up storage permissions
Tip: Always update Termux packages before installing new tools.

Lesson Notes
